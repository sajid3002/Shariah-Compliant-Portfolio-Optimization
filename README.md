# Shariah-Compliant-Portfolio-Optimization
This Shariah-compliant portfolio optimization project analyzes AAPL, NVDA, JNJ, ASML, GLD, and SLV. Using Python, it compares Tangency (Sharpe: 1.51), GMV (Vol: 12%), and 1/N strategies. It demonstrates how mean-variance optimization balances high-growth tech with gold anchors to achieve "Hifz al-Mal" (wealth preservation) and efficiency.
